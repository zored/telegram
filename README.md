# PHP Telegram API
Subset of Telegram API that you are free to extend.

## Features
- Object oriented.
- Based on PHP7.1.
- Easy to extend.
- Easy to use.

## Example
- Copy `.env.dist` into `.env` and fill your values.
-[Warm-up session cache](./bin/warmup.php) (optional, but first run takes time). 
- Run:
    - [Client](./bin/client.php)
    - [Server](./bin/bot.php)
